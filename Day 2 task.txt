1. Difference betwen Document and Window objects?  

Document object:
  It represent a web page that is loaded in the browser.
  It is loaded inside the window.
  It is the object of the window property.
  Through this we can access the element in the HTML page, can add dynamic content to our web page.
  Property of document objects can also be accessed by a window.
  All tags, elements with attributes in HTML are part of the document.
   

Window object:
     It is the topmost object of the DOM hierarchy.
     It represent a browser window that displays the content of the webpage.
     It is the object of the browser.
     It is created whenever a window appear on the screen to display the content of the document.
     Property of the window object cannot be accessed by document object.
     Global objects, functions,variables of JS are memmber of the window object.
